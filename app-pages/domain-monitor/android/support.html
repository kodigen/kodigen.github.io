In the main screen, at the bottom right there must be a button with a plus sign (+) inside. Press the button then write the domain name i.e. "google.com" and press "Add".

There can be so many errors to prevent you to adding a domain.

If there is no specific error, please check possible errors below:

  • A trouble in your internet connection.
  • WHOIS server which is you requested is blocks you due too many requests.
  • Your ISP or VPN provider is blocking WHOIS requests.
  • Corruptions on WHOIS data. You can notify us about this.

Not all domain extensions may supported. Some domain extensions like ".de" doesn't provide expiry date on their own WHOIS data.

For Domain Monitor version >= 0.10

By default, Domain Monitor doesn't show a sticky notification until you enable "strict monitoring mode". You can disable the strict monitoring anytime from settings.


For Domain Monitor version <= 0.9

Android 8.0 and above versions are requires sticky notifications to keep background services running for security reasons. The notification system is dependent this background service.

Domain Monitor has a background service to monitor the changes on the domain, naturally. The domains you've added will be scanned in order. The monitoring service will continue running in the background as long as you keep it enabled. You can disable it anytime from settings if you don't want to be notified.

For Domain Monitor version >= 0.10

Strict monitoring can be enabled from settings. If it's enabled there will be a sticky notification that requiring by the Android system to monitor in consistent. If you are a domain catcher, it is recommended.

Without strict monitoring, you can monitor domains regularly but without consistent interval. It's enabled by default and no sticky notification will shown.

Domain Monitor doesn't use any third party services to obtain the domain availability or expiry date information. Requests are sent by directly from your device. But we're using Google Analytics, to analyze the user behaviour. You can reach our privacy policy and Google privacy policy.
